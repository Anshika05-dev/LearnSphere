# LearnSphere

**LearnSphere** is a full-stack LMS (Learning Management System) project built with modern web technologies.  
It provides both a **frontend user interface** and a **backend API/services** that together enable student and instructor workflows for an education platform.

This repository contains both the **frontend** and **backend** of LearnSphere.
